### Hi there 👋


My name is Martin and I'm a researcher at the [Lundbeck Centre for GeoGenetics](https://globe.ku.dk/research/geogenetics/), which is part of the [GLOBE Institute](https://globe.ku.dk/) at the University of Copenhagen.

My background is in ancient DNA genomics. I did my PhD [Max Planck Institute for Evolutionary Anthropology](https://www.eva.mpg.de/genetics/index/) in Leipzig where I worked on the evolutionary histories of modern humans, Neanderthals, and Denisovans. My PhD advisors were [Janet Kelso](https://en.wikipedia.org/wiki/Janet_Kelso) and [Svante Pääbo](https://en.wikipedia.org/wiki/Svante_P%C3%A4%C3%A4bo). Before I moved to Leipzig to do my PhD, I did an undergraduate degree in computer science at Charles University in Prague.

In my current job, I’m working on spatiotemporal methods for population genetics, ecology and evolution. The most recent outcome of this work is an R package called [_slendr_](https://github.com/bodkan/slendr) which provides an easy to use framework for building programmable population genetic simulations on a real geographic landscape without any geospatial know-how.

I like thinking about ways to make the interface between us and our computational tools as streamlined and smooth as possible. My favourite programming language is R. Not because it's perfect, but because it makes it exceptionally easy to build elegant interfaces to other tools even when these are, on their own, much less user-friendly than would be desirable. This path has started with me building a modest R interface for the widely used ADMIXTOOLS software, implemented in the R package [_admixr_](https://github.com/bodkan/admixr).

You can find more information about me, my work, and about the ways to contact me on my [personal website](https://bodkan.net/).
